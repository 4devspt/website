# 4devs.pt

Projeto open source alojado em [4devs.pt](https://4devs.pt).

O objetivo é criar utilitários para desenvolvedores.

Se quiseres adicionar algo, sente-te à vontade para criar uma Pull Request.

## Como executar localmente

1. Instala o Angular CLI globalmente:
   ```bash
   npm install -g @angular/cli
   ```

2. Instala as dependências:
   ```bash
   npm install
   ```

3. Inicia o servidor de desenvolvimento:
   ```bash
   npm start
   ```

4. Abre o navegador em `http://localhost:4200`
